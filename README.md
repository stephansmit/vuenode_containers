# NodeJS container 

Container with [Vue](https://vuejs.org)

## Build the container locally
~~~~
singularity build vuenode_containers.sif Singularity
~~~~

## Pull the container
~~~~
singularity pull shub://stephansmit/vuenode_containers:latest
~~~~
